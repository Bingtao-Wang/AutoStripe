# 实验记录说明

## 使用方法

1. **开始新实验前**，复制 `experiment_template.md` 并重命名：
   ```bash
   cp experiment_template.md exp_001_town05_clear.md
   ```

2. **运行实验**，按E键开始/停止记录

3. **实验结束后**，填写模板中的各项内容：
   - 基本信息：从命令行参数和运行日志获取
   - 实验结果：从生成的CSV文件获取（`slam_eval_summary.csv`）
   - 观察与分析：根据实际观察填写
   - 可视化：截图保存到实验文件夹

4. **数据整理**：
   - 每个实验的数据保存在 `run_YYYYMMDD_HHMMSS/` 文件夹
   - CSV文件包含详细的pose数据和评估指标
   - 轨迹图自动生成

## 实验编号规范

建议使用以下命名格式：
- `exp_001_town05_clear.md` - Town05地图，晴天
- `exp_002_town01_night.md` - Town01地图，夜晚
- `exp_003_town03_rain.md` - Town03地图，雨天

## 数据分析建议

1. **对比不同场景**：同一地图不同天气条件
2. **对比不同地图**：Town01（城市）vs Town05（高速）
3. **长时间测试**：评估累积漂移
4. **初始化测试**：记录不同运动模式下的初始化成功率
